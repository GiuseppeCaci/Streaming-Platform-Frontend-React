# Streaming Platform – Frontend

## Description
This project is the **frontend application** of a full-stack streaming platform prototype.

The interface was built using **React** and communicates with a custom backend API to manage users, authentication, and streaming content.

The goal of this project was to simulate the architecture of a modern web platform with a client-server structure.

## Tech Stack
- React
- JavaScript
- CSS
- REST API
- JWT Authentication

## Features
- User authentication interface
- Communication with backend API
- Dynamic content rendering
- Component-based UI architecture

## Architecture
This repository contains the **frontend client** of the platform.

The backend API is available in the companion repository:
`Streaming Platform – Backend`

## Notes
This project was developed as a full-stack learning project to explore modern web application architecture.
The live demo is currently offline, but the full source code is available for review.
